This program is a simplfied blackjack game.  To run it, compile and run blackjack.cpp or execute blackjack.exe.  
Once the game is running, select 1 or 2 to have the computer make random decisions or informed decisions respectively.
When it is your turn follow the instructions and select h for hit, s for stand, or p for split.  Once the game ends, 
select Y or N to exit.
